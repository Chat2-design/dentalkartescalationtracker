# Dentalkart Escalation Tracker

## Deploy to Netlify (1-click)

### Option 1: Netlify Drop (easiest - no account needed)
1. Go to https://app.netlify.com/drop
2. Drag and drop the `index.html` file onto the page
3. Your tracker is live instantly!

### Option 2: Netlify with account (for a permanent URL)
1. Create a free account at https://app.netlify.com
2. Click "Add new site" → "Deploy manually"
3. Drag the folder or zip file
4. Your site will be at `https://your-site-name.netlify.app`

### Option 3: GitHub + Netlify (for team sharing)
1. Push this folder to a GitHub repo
2. Connect repo to Netlify
3. Auto-deploys on every push

## Features
- Add/edit/delete escalation cases (Bluedart & Delhivery)
- Dashboard with live stats and charts
- TAT tracking (email sent → close date)
- Highest delay case highlighted
- CSV export
- Share link with team
- All data stored in browser localStorage
